# 1003 EPK

Projet prêt pour GitHub Pages.

## Publication

1. Crée un dépôt GitHub public, par exemple `1003-epk`.
2. Upload `index.html`, `style.css` et `script.js` à la racine du dépôt.
3. Va dans Settings → Pages.
4. Source : Deploy from a branch.
5. Branch : main / root.
6. Sauvegarde.

URL finale :
`https://TON-PSEUDO.github.io/1003-epk/`

## Modifications rapides

- Textes : modifier `index.html`.
- Design : modifier `style.css`.
- Fenêtre WhatsApp : modifier `script.js`.
